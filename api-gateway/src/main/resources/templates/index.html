Swagger UI

APIs for each VRO tenant is available via VRO's Swagger UI.
  • Select the tenant from the "Select a definition" dropdown on the upper-right of the UI.
  • Each tenant page has a link to the API spec.
  • To use Swagger's "Try It" feature, make sure to select the correct server on the left dropdown. For example, for the VRO App, select the "/app - via API Gateway" server.
